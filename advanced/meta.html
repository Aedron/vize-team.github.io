# 页面元数据

页面的基础元数据，都可以通过 meta 属性在物料中等处获取。

也可以在其他地方通过 window.VIZE.meta 获取。

Meta 信息包含的如下字段：

  • title：页面标题 string
  • desc：页面描述 string
  • id：页面 id number | undefined
  • key：页面 key string
  • isTemplate：是否为模板页面 boolean
  • isEditor：是否为编辑器环境 boolean
