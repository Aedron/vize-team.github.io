# 介绍

Vize：代化的 Web 可视化搭建引擎

Github: https://github.com/vize-team/vize (opens new window)

# ✨ 特性

  • 友好、便捷的可视化编辑体验
  • 完全所见即所得
  • 丰富的物料抽象粒度
  • 支持物料间联动，强大的可配置事件系统
  • 物料与系统完全解耦
  • 完备的开发工具链支持
  • 便捷的接入方式
